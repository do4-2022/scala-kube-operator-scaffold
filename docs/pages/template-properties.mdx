# Template properties

This is a list of all the properties that can be configured in the template:

| Property                              | Description                                    | Default            |
| ------------------------------------- | ---------------------------------------------- | ------------------ |
| `name`                                | The name of the operator                       | `my-operator`      |
| `organization`                        | The organization name                          | `com.organization` |
| `example_resource_group_organization` | The organization name for the example resource | `com.organization` |
| `example_resource_group_extension`    | The extension name for the example resource    | `com`              |
| `example_resource_name_kind`          | The kind name for the example resource         | `MyResource`       |
| `example_resource_name_singular`      | The singular name for the example resource     | `myresource`       |
| `example_resource_name_plural`        | The plural name for the example resource       | `myresources`      |

If don't understand what these properties mean, you can open an issue and we will try to explain them better.
